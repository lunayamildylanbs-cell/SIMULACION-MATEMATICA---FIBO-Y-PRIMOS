03

Explicación de los Algoritmos

Algoritmo de Fibonacci

La secuencia Fibonacci se genera sumando los dos números anteriores. El algoritmo utiliza variables simples y ciclos repetitivos. 

let a = 0;
let b = 1;
let c;

for (let i = 1; i <= meses; i++) {
   c = a + b;
   a = b;
   b = c;
}

Algoritmo de números primos

Un número primo solamente puede dividirse exactamente entre 1 y sí mismo. El algoritmo verifica cuántos divisores tiene el número. 

let contador = 0;

for (let i = 1; i <= numero; i++) {

   if (numero % i == 0) {
      contador++;
   }

}

if (contador == 2) {
   // El número es primo
}
