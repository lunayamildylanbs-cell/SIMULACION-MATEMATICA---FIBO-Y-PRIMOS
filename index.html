Explicación de los Algoritmos

Algoritmo de Fibonacci

La Serie de Fibonacci se genera sumando los dos números anteriores. Esta secuencia aparece en distintos fenómenos naturales y también puede utilizarse para representar crecimientos progresivos.

En este proyecto el algoritmo se utiliza para simular crecimiento de ahorro, expansión de datos y secuencias matemáticas progresivas. 

let a = 0;
let b = 1;
let c;

for(let i = 1; i <= meses; i++){

   c = a + b;
   a = b;
   b = c;

}

Algoritmo de Números Primos

Un número primo es aquel que solamente puede dividirse exactamente entre 1 y sí mismo.
Este algoritmo analiza cuántos divisores exactos posee un número. Si solamente tiene dos divisores, el sistema determina que el número es primo.

Los números primos son ampliamente utilizados en criptografía, seguridad digital y protección informática. 

let contador = 0;

for(let i = 1; i <= numero; i++){

   if(numero % i == 0){
      contador++;
   }

}

if(contador == 2){

   // Es primo

}

📈

Simulación de crecimiento financiero

Aplicación de Fibonacci en economía personal.

Una cooperativa estudiantil desea implementar un sistema de ahorro progresivo para jóvenes universitarios.

El problema principal es que muchos estudiantes abandonan sus planes de ahorro debido a depósitos demasiado elevados o sistemas poco dinámicos.

Para resolver este problema, se propone utilizar la Serie de Fibonacci como modelo matemático de crecimiento progresivo.

En lugar de ahorrar una cantidad fija, cada depósito mensual aumentará siguiendo la secuencia Fibonacci, permitiendo observar cómo pequeñas cantidades pueden transformarse en un ahorro considerable.

Problema 2 · Sistema de Seguridad Digital

🔐

Decodificación y validación de claves

Uso de números primos en seguridad informática.

Un laboratorio tecnológico necesita crear un sistema básico de validación matemática para códigos de acceso.

El objetivo es analizar si un número puede considerarse suficientemente seguro para utilizarse como parte de una clave digital.

Para ello se utilizarán números primos, ya que muchos sistemas modernos de criptografía emplean propiedades matemáticas relacionadas con divisiones y factorización prima.

El simulador verificará automáticamente cuántos divisores exactos posee el número ingresado.

Si solamente puede dividirse entre 1 y sí mismo, el sistema lo considerará un código matemáticamente válido.

Problema 3 · Análisis Matemático Avanzado

🧠

Fibonacci y Primos combinados

Detección matemática automática.

Un centro de análisis computacional necesita investigar cuáles valores de la secuencia Fibonacci también pueden considerarse números primos.

Este tipo de análisis matemático es utilizado en teoría numérica, optimización computacional, análisis de algoritmos y estructuras avanzadas de datos.

El sistema generará automáticamente la secuencia Fibonacci y luego analizará cada número individualmente.

Después del análisis, el simulador mostrará cuáles términos también son números primos, permitiendo observar relaciones matemáticas complejas dentro de una misma secuencia.
